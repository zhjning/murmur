Summary of Bioinformatics Resources

Mar 27, 2020 · 1 min read
Data analysis books: gencore single-cell analysis Running Tricky Tools 🏃🏃🏃 Review tools: Carrot 2: 可以可视化关键词的关联词汇，比自己手动分类要便捷很多，关键文献都有，但不太全，需要自己后面再检索添加。另外参数调整也会影响分类结果，在前期检索过程中，最好多调整几次参数，有助于获得更加全面的关联词汇表（如检索VB时，限制150篇文献时可以找到T2M为VB的关联词汇，限制500篇时，T2M相关文献会被归到其他关联词中，不能汇聚到T2M概念下） Zotero: 文献管理工具，支持Word插入，可以直接搜索文献名中的关键词插入，比EndNote用户体验要好一万倍！！！有chrome插件，浏览需要的文献时，可以直接同步保存到账户，本地pdf拖入也非常方便，不需要花大量时间编写meta信息，软件会直接提取管理。就是免费版多账户同步有存储容量限制。
